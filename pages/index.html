Introduction

This will become a collection of open and free endpoints which can be used with Tana. Created and maintained by https://twitter.com/BasB3.

1. YouTube transcript endpoint

API requests should be made to the following base URL: https://tana-tools.bb-projects.nl/youtube_transcript

Endpoints

GET /youtube_transcript

Retrieve the user's information by their unique identifier.

URL Parameters

  • youtube_url - YouTube URL (any format) or YouTube id both work.

Example Request

GET https://tana-tools.bb-projects.nl/youtube_transcript?youtube_url=https://www.youtube.com/watch?v=yu7n0XzqtfA&pp=ygUPbWFyY3VzIGF1cmVsaXVz

Response

Raw text with the youtube transcript if available.
